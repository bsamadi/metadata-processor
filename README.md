# metadata-processor
This repository contains a Flask implementation of a REST API to process image metadata.  

# Make commands

- `make DOCKER_USER=bsamadi docker-build`
- `make DOCKER_USER=bsamadi docker-push`
- `make DOCKER_USER=bsamadi docker-pull`
